# Urna Eletrônica Simulada - Exemplo Demonstrativo

Este projeto tem como objetivo **demonstrar, de forma totalmente fictícia**, um possível cenário onde **um candidato é favorecido de forma fraudulenta** em uma simulação de urna eletrônica. Ele foi criado **apenas para fins educacionais, satíricos e de reflexão** sobre como manipulações em sistemas eletrônicos podem impactar resultados.

## ⚠️ Aviso Importante

> **Este projeto NÃO representa a realidade das eleições brasileiras**.  
> Ele é **uma simulação hipotética** inspirada em discursos e teorias que circularam nas redes sociais.  
> **Não é baseado em fatos ou dados oficiais.**

---

## 🧠 Sobre o Projeto

A interface simula uma urna eletrônica onde o eleitor digita o número de seu candidato (13 ou 22), podendo votar em **Lula** (13) ou **Bolsonaro** (22). No entanto, há uma manipulação oculta no código que favorece o candidato 13, mesmo quando o voto é direcionado ao 22.

### Exemplos de manipulação no código:

- Votos para "Branco" também contam como votos para o candidato 13.
- A cada **2 votos válidos para o candidato 22**, **1 voto é automaticamente atribuído ao candidato 13**.
